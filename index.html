Create your own dances using DNCE

About the Project

I created DNCE mainly for fun, you can create and your own dance sequences and see how they would look. For a more practical purpose, you can quickly create a dance and send it to others for them to learn. Through this language, you can also learn what certain dance moves are called and what they look like.

Features

  • Feature 1: Outputs a gif of the series of steps you created.
  • Feature 2: Very simple syntax.
  • Feature 3: You get to choose a style and a name for your dance.

If you'd like to learn more, feel free to explore the repository on GitHub.
